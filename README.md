# 💰 MiFinanzas — Dashboard de Finanzas Personales

> **Versión 1.0** · Aplicación web para la gestión inteligente de tu dinero personal.

---

## 🌟 ¿Por qué es importante administrar tu dinero?

La administración del dinero no es solo una habilidad financiera — **es una habilidad de vida**. Estudios demuestran que el 78% de las personas que no llevan un control de sus finanzas terminan viviendo de quincena en quincena, sin importar cuánto ganen.

### La realidad financiera

- 📉 **Sin control, no hay crecimiento.** Gastar sin registrar es como conducir sin mapa: puedes avanzar, pero no sabes hacia dónde.
- 💸 **Los gastos pequeños se acumulan.** Ese café diario, la suscripción que olvidaste cancelar... al final del mes, pueden representar una fuga significativa.
- 🎯 **Las metas necesitan estructura.** Ya sea ahorrar para un viaje, un fondo de emergencia o invertir, todo empieza con saber exactamente cuánto entra y cuánto sale.
- 🧠 **La tranquilidad mental tiene precio.** El estrés financiero afecta tu salud, tus relaciones y tu productividad. Tener claridad sobre tus finanzas elimina esa ansiedad.

> *"No se trata de cuánto dinero ganas, sino de cuánto conservas, cómo lo pones a trabajar y para cuántas generaciones lo mantienes."* — Robert Kiyosaki

## 🚀 ¿Cómo te ayuda MiFinanzas?

**MiFinanzas** fue creada con una filosofía simple: **hacer visible lo invisible**. Tu dinero se mueve todos los días, y esta aplicación te permite ver exactamente hacia dónde va.

### ✅ Lo que puedes hacer

| Funcionalidad | Descripción |
|---|---|
| 🏦 **Gestión de 4 cuentas** | Controla tu Cuenta Bancaria 1, Cuenta Bancaria 2, Alcancía y Cartera de forma independiente |
| 📈 **Registro de ingresos** | Clasifica tus entradas por origen: Sueldo Fijo, Proyecto de Publicidad, Venta de App, Prestaciones y más |
| 📉 **Registro de gastos** | Categoriza cada gasto: Comida, Plan Móvil, Transporte, Entretenimiento, Servicios |
| 📊 **Dashboard visual** | Saldo total consolidado + tarjetas individuales por cuenta con actualización en tiempo real |
| 📜 **Historial completo** | Lista detallada de todas las transacciones con filtros por tipo y cuenta |
| 💾 **Datos persistentes** | Tus datos se guardan en el navegador (localStorage) — no se pierden al recargar |
| 📱 **Diseño responsive** | Funciona perfectamente en móviles, tablets y escritorio |

### 🎨 Diseño Premium

- Interfaz oscura moderna estilo Dashboard profesional
- Colores intuitivos: **verde** para ingresos, **rojo** para gastos
- Animaciones suaves y micro-interacciones
- Tipografía Inter para máxima legibilidad

## 🚀 Instalación y Uso

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/yosephdev504-del/MiFinanzas.git
   ```

2. **Abre la aplicación:**
   ```bash
   cd MiFinanzas
   # Abre index.html en tu navegador favorito
   ```

3. **¡Listo!** No necesitas servidor, ni npm, ni configuración alguna.

---

## 📋 Guía Rápida

1. **Configura tus cuentas** → Ve a la sección "Cuentas" y establece el saldo inicial de cada una
2. **Registra ingresos** → Cada vez que recibas dinero, regístralo indicando origen y cuenta destino
3. **Registra gastos** → Cada compra o pago, regístralo con su categoría y cuenta fuente
4. **Consulta tu dashboard** → Visualiza en tiempo real el estado de todas tus finanzas
